                                                            HOSPITAL MANAGEMENT SYSTEM
                                                Command Line Application using Java, JDBC and MySQL

PROJECT OVERVIEW :-

This Hospital Management System is a command line based Java application
designed to manage basic hospital operations such as patient registration,
doctor management and appointment scheduling.

The project is developed using Java with JDBC for database connectivity and
MySQL as the backend database. The main goal of this project is to understand
core backend concepts such as database interaction, layered architecture,
CRUD operations and clean code structure.

This project is suitable for Java beginners, backend learners, academic
projects and GitHub portfolio demonstration.

TECHNOLOGY STACK :-

Programming Language  : Java  
Database              : MySQL  
Database Connectivity : JDBC  
Application Type      : Command Line Interface (CLI)  
Architecture Pattern  : Layered Architecture  

FEATURES INCLUDED :-

Doctor Management
• Store doctor details such as name, specialization and contact number
• Doctors are used for appointment scheduling

Patient Management
• Register new patients
• Store patient details like name, age, gender and phone number
• View all registered patients from database

Appointment Management
• Book appointments between doctors and patients
• Store appointment date and relationship
• View all appointments
• Uses foreign key constraints for data consistency

System Features
• JDBC based database connection
• Proper exception handling
• Modular and readable code
• Easy to extend and maintain
